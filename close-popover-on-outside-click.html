jquery .has() method : The has() method returns all elements that have one or more elements inside of them, that matches the specified selector.

Tip: To select elements that have multiple elements inside of them, use comma (see example below).

// Example
// Return all <p> elements that have a <span> element inside of them:

// Syntax : $(selector).has(element)

$("p").has("span")

jQuery is() Method

is() Checks the set of matched elements against a selector/element/jQuery object, and return true if at least one of these elements matches the given arguments. The is() method checks if one of the selected elements matches the selectorElement.

// Example
// If the parent of <p> is a <div> element, alert some text:

// Syntax : $(selector).is(selectorElement,function(index,element))

if ($("p").parent().is("div")) {
    alert("Parent of p is div"); 
}
